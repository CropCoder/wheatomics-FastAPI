PfamSearch — Search Genes by PFAM Domain

Must start with "PF". Exact match only — PF0031 will also match PF00319 etc. (regex search).
All tables from Genefuncdb. Only tables with a Domain column support PFAM search.
About PfamSearch: This tool searches all wheat genes annotated with a specific PFAM domain. Results include chromosomal positions (in Mb), strand orientation, gene ID, functional description, and domain ID. The database (Genefuncdb) uses regular expression matching on the Domain field, so PF0031 will match PF00310, PF00319, PF003100 etc.
Corresponding to: protein family search (original CGI version)

Examples:

  • PF00319 — SRF-type transcription factor (MADS-box)
  • PF00830 — Ribosomal protein L14p/L23e
  • PF00931 — NB-ARC domain (disease resistance)
  • PF00203 — Ribosomal protein S19
  • PF05699 — DUF820
  • PF03018 — Dirigent protein
  • PF07160 — DUF1393
  • PF02182 — SAD1/UNC-like domain
