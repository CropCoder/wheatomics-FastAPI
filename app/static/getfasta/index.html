GetSequence — Retrieve FASTA by Gene ID, Batch IDs, or Interval

Enter gene ID(s) or a chromosomal interval (chr_name:start-end). Auto-detected.
Examples:
  • TraesCS5A02G391700 — VRN-A1, single gene
  • 3 wheat genes, batch mode
  • Chr1A_Chinese_Spring1.0:200-500 — Chinese Spring v1.0 interval
  • Chr1A_Abo:200-500 — Abbondanza interval
Defaults to all_gene (CDS), or pick a genome-specific database for interval queries.
