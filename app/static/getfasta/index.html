GetSequence — Retrieve FASTA by Gene ID, Batch IDs, or Interval

Single gene ID, e.g. TraesCS5A02G391700
Defaults to all_gene (CDS). Use all_protein for protein, or a genome-specific database for interval queries.
Defaults to all_protein. Must use a protein BLAST database, not a gene/genome one.
Examples:
  • TraesCS5A02G391700 — VRN-A1, single gene (CDS+protein)
  • 3 wheat genes, batch mode
  • chr1A:1-141522 — Chinese Spring v1.0 interval
  • Chr1A_Abo:200-500 — Abbondanza interval
